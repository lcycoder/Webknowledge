# 判断数据类型
## typeof
1. typeof返回的类型有七种:boolean,number,string,undefined,object,function,symbol
(以下是特殊的情况)
2. number类型: NAN,infinity 
3. object类型:对象,数组,null 

## instanceof
* 简单来说: 就是判断是否是某个构造函数的实例化对象
例如
```js
function Person(){}
var p = new Person();
p instanceof Person
```
* null 用instanceof判断的话 不为object
* instanceof不能判断基本类型数据
