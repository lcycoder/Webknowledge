# 排序方法
## 冒泡排序
```js
    function bubblesort(arr){
        let len = arr.length-1;
        for(let=0;i<len;i++){
            for(let j=0;j<len-i>;j++){
                if(arr[j]>arr[j+1]){ //大于则交换两个的位置
                    let temp = arr[j];
                    arr[j] = arr [j+1];
                    arr[j+1] = temp;
                }
            }
        }
    }
```
