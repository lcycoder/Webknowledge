## 冒泡排序
* 两个数据依次比较,每次将最大的数值放到最后
![冒泡](https://pic2.zhimg.com/v2-33a947c71ad62b254cab62e5364d2813_b.jpg)
```js
    function bubblesort(arr){
        let len = arr.length-1;
        for(let=0;i<len;i++){
            for(let j=0;j<len-i>;j++){
                if(arr[j]>arr[j+1]){ //大于则交换两个的位置
                    let temp = arr[j];
                    arr[j] = arr [j+1];
                    arr[j+1] = temp;
                }
            }
        }
    }
```
