# bootstrap
* 响应式具有的特点
    * 网页宽度自动调整
    * 尽量少用绝对宽度
    * 字体用rem，em做单位
## 栅格系统  
* 排列
    * col-xl-2
    * col-lg-3 
    * col-md-4
    * col-sm-6
    * col-12
* 对齐
    * 垂直对齐
        * 行的对齐方式
            * align-items-start 顶对齐
            *             center 中间对齐
            *             end    底对齐
        * 列的单独对齐方式
            * align-self-start  
            *            center
            *            end 
    * 水平对齐
        * justify-content-start 左对齐
        *                 center 居中对齐
        *                 end 右对齐
        *                 around 分散居中对齐（每个元素两侧的间距是平等的）
        *                 between 左右两端对齐（元素之间的间距是自动平分的）
    * 列偏移
        * 
    * 间距
        * mr-{breakpoint}-auto 使右侧的列远离到最右边
        * ml-{breakpoint}-auto 使左侧的列远离到最左边
