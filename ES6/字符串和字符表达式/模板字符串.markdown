var text = "登哥喜欢秋葵\n"
# 模板字符串
* es提供了简单的表示方法
    1. 实现字符串的换行,只需要在字符串开头和结尾加` 符号即可
    2. 实现字符串拼接,使用${},括号内写变量的名字,也可以写表达式,也可以嵌套.
    ```js
        var i = ':';
        var str = `name${i}
        lichengyang`    /* name:  
        lichengyang */
        
    ```
